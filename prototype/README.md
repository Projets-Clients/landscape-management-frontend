# Prototype LANDSCAPE

Interactive design prototype imported from Claude Design.

## Running

Serve this directory with any static HTTP server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

Then open http://localhost:5000 (or 8080) in your browser.

## Source

Claude Design project: https://claude.ai/design/p/22783e59-e9b1-4b2e-b5df-595a8ef8af13

## Missing files

The following screen files (screens-*.jsx, components.jsx, components.css, tweaks-panel.jsx, image-slot.js)
need to be downloaded from the Claude Design project above and placed in this directory.
